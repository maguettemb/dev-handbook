# 6. Déploiement

Faire passer une application du poste de dev à un environnement accessible et fiable en production.

## Ce que couvre cette section

- **[Kubernetes — les essentiels](kubernetes.md)** — pods, services, déploiements, sans rentrer dans l'administration complexe
- **[Argo CD & GitOps](argocd.md)** — déployer en synchronisant l'état désiré depuis Git

## Approche

Cette section reste volontairement au niveau "ce qu'un AI/ML engineer doit comprendre pour collaborer avec une équipe infra", pas "devenir SRE". L'objectif est de lire un manifeste Kubernetes ou une pipeline Argo CD et de comprendre ce qui se passe.
