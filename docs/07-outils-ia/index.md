# 7. Outils IA pour développer

Les assistants de code font désormais partie du workflow standard. Cette section documente un usage réfléchi — pas juste "ça génère du code".

## Ce que couvre cette section

- **[Comparatif : Claude Code vs Copilot vs Cursor](comparatif.md)** — pour/contre selon le cas d'usage

## Point de vigilance

Un outil IA de code est un accélérateur, pas un substitut à la compréhension du code produit. Cette section documente aussi les pièges rencontrés : dépendance excessive, code généré non revu, prompts mal calibrés.
